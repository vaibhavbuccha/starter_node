# Loops in Node js
---
#### In node js 4 types of loops are available
- for
- for/in
- for/of
- while
- do/while

1. for Loop

```js
for (statement 1; statement 2; statement 3) {
  // code block to be executed
}
```
example

```js
for (i = 0; i < 5; i++) {
  console.log(i);
}

/**
 * 
 * Result
 * 0
 * 1
 * 2
 * 3
 * 4
 * 
*/
```
